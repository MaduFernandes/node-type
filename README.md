## Install

```sh
npm install
```

## Usage

Run migrations:

```sh
npm run knex:migrate
```

Run seed:

```sh
npm run knex:seed
```

Run the application:

```sh
npm run dev
```

## Tests

Run the tests:

```sh
npm run jest
```
